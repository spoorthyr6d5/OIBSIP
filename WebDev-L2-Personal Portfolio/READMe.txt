# Spoorthy Racharla | Personal Portfolio Website

A clean, modern, and fully responsive personal portfolio website built with semantic HTML5 and CSS3. This project serves as a digital resume and project showcase for **Spoorthy Racharla**, an Aspiring Web Developer and Computer Science Student specializing in AI & ML.

---

## ✨ Features

Hero Section**: A welcoming introduction featuring a profile image, professional headlines, and quick access to a downloadable resume

Education & Career Journey**: Timeline layout outlining academic history from high school to a B.Tech in CSE (AI & ML)

Interactive Skills Grid**: A neat, scannable display of current technical proficiencies (Languages, Frameworks, and Tools)

Project Showcase**: Highlights key academic and self-paced projects, emphasizing practical applications of AI, ML, and web technologies

Professional Milestones**: Dedicated sections detailing industry certifications (IBM, Cisco, JP Morgan) and hands-on internship experience.

Contact Hub**: Accessible, direct links to professional social channels (LinkedIn, GitHub) and email communication
