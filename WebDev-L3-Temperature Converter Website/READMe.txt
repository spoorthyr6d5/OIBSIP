# Temperature Converter

A clean, minimalist, and responsive web application built to convert temperatures instantly between various standard units.

---

## ✨ Features

Multi-Unit Support**: Seamlessly converts temperatures between **Celsius (°C)**, **Fahrenheit (°F)**, and **Kelvin (K)**.

Dynamic Form Fields**: Built with simple dropdown selection menus (`<select>`) for quick toggling between source and target units.

Responsive Interface**: Features a centralized card component (`.card`) that scales nicely to fit mobile screens and desktops alike.

Live Results Placeholder**: A dedicated output display box (`#result`) providing immediate feedback once calculations are triggered.
